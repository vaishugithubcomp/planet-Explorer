> **"Embark on a cosmic journey through our Solar System!"**

`Planet Explorer` is a stunning web-based interactive simulation that brings the planets of our solar system to life. From vibrant visuals to real-time data, this project turns learning into an engaging space adventure.

## 💡 Features

- ✨ Stylish welcome screen with fade-in animations
- 🪐 Interactive **planet cards** with hover effects
- 🔍 **Info panel** with real planetary data
- 📏 Shows distance between planets dynamically
- 🎨 Universe-themed responsive UI
- 🖱️ Hover effects, glowing buttons, and planet rotations
